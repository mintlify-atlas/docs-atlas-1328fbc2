## Overview

Testing is critical for maintaining code quality and ensuring Clean Architecture boundaries are respected. This guide covers testing strategies for each layer of the application.

<Info>
  The project is configured with `flutter_test` for unit and widget testing.
</Info>

## Test Environment Setup

### Dependencies

The following test dependencies are configured in `pubspec.yaml`:

```yaml pubspec.yaml (dev_dependencies)
dev_dependencies:
  flutter_test:
    sdk: flutter
  flutter_lints: ^3.0.0
  build_runner: ^2.4.8
```

### Running Tests

<CodeGroup>

```bash Run All Tests
flutter test
```

```bash Run Specific Test File
flutter test test/features/product/domain/usecases/get_product_test.dart
```

```bash Run with Coverage
flutter test --coverage
```

```bash Run in Watch Mode
flutter test --watch
```

</CodeGroup>

<Tip>
  Use `flutter test --watch` during development for instant feedback on code changes.
</Tip>

## Testing by Layer

### 1. Domain Layer Testing

Domain layer tests are the **most important** as they verify business logic without framework dependencies.

<Tabs>
  <Tab title="Entity Tests">
    ```dart test/features/billing/domain/entities/cart_item_test.dart
    import 'package:flutter_test/flutter_test.dart';
    import 'package:billing_app/features/billing/domain/entities/cart_item.dart';
    import 'package:billing_app/features/product/domain/entities/product.dart';

    void main() {
      group('CartItem', () {
        const testProduct = Product(
          id: '1',
          name: 'Test Product',
          barcode: '12345',
          price: 10.0,
          stock: 100,
        );

        test('should calculate total correctly', () {
          const cartItem = CartItem(
            product: testProduct,
            quantity: 3,
          );

          expect(cartItem.total, 30.0);
        });

        test('should default quantity to 1', () {
          const cartItem = CartItem(product: testProduct);

          expect(cartItem.quantity, 1);
        });

        test('copyWith should update only specified fields', () {
          const cartItem = CartItem(
            product: testProduct,
            quantity: 2,
          );

          final updated = cartItem.copyWith(quantity: 5);

          expect(updated.quantity, 5);
          expect(updated.product, testProduct); // Unchanged
        });

        test('should support value equality via Equatable', () {
          const item1 = CartItem(product: testProduct, quantity: 2);
          const item2 = CartItem(product: testProduct, quantity: 2);
          const item3 = CartItem(product: testProduct, quantity: 3);

          expect(item1, item2);          // Equal
          expect(item1, isNot(item3));   // Not equal
        });
      });
    }
    ```
  </Tab>
  
  <Tab title="Use Case Tests">
    ```dart test/features/product/domain/usecases/get_product_by_barcode_test.dart
    import 'package:flutter_test/flutter_test.dart';
    import 'package:fpdart/fpdart.dart';
    import 'package:mocktail/mocktail.dart';
    import 'package:billing_app/core/error/failure.dart';
    import 'package:billing_app/features/product/domain/entities/product.dart';
    import 'package:billing_app/features/product/domain/repositories/product_repository.dart';
    import 'package:billing_app/features/product/domain/usecases/product_usecases.dart';

    class MockProductRepository extends Mock implements ProductRepository {}

    void main() {
      late GetProductByBarcodeUseCase useCase;
      late MockProductRepository mockRepository;

      setUp(() {
        mockRepository = MockProductRepository();
        useCase = GetProductByBarcodeUseCase(mockRepository);
      });

      const testBarcode = '12345';
      const testProduct = Product(
        id: '1',
        name: 'Test Product',
        barcode: testBarcode,
        price: 10.0,
        stock: 100,
      );

      test('should return product when repository call succeeds', () async {
        // Arrange
        when(() => mockRepository.getProductByBarcode(testBarcode))
            .thenAnswer((_) async => const Right(testProduct));

        // Act
        final result = await useCase(testBarcode);

        // Assert
        expect(result, const Right(testProduct));
        verify(() => mockRepository.getProductByBarcode(testBarcode)).called(1);
      });

      test('should return CacheFailure when product not found', () async {
        // Arrange
        const failure = CacheFailure('Product not found');
        when(() => mockRepository.getProductByBarcode(testBarcode))
            .thenAnswer((_) async => const Left(failure));

        // Act
        final result = await useCase(testBarcode);

        // Assert
        expect(result, const Left(failure));
        verify(() => mockRepository.getProductByBarcode(testBarcode)).called(1);
      });
    }
    ```
  </Tab>
</Tabs>

<Card title="Testing Principle" icon="lightbulb">
  Use cases should be tested with **mocked repositories** to isolate business logic.
</Card>

### 2. Data Layer Testing

<Tabs>
  <Tab title="Model Tests">
    ```dart test/features/product/data/models/product_model_test.dart
    import 'package:flutter_test/flutter_test.dart';
    import 'package:billing_app/features/product/data/models/product_model.dart';
    import 'package:billing_app/features/product/domain/entities/product.dart';

    void main() {
      group('ProductModel', () {
        const testProductModel = ProductModel(
          id: '1',
          name: 'Test Product',
          barcode: '12345',
          price: 10.0,
          stock: 100,
        );

        test('should be a subclass of Product entity', () {
          expect(testProductModel, isA<Product>());
        });

        test('fromEntity should create ProductModel from Product', () {
          const product = Product(
            id: '1',
            name: 'Test Product',
            barcode: '12345',
            price: 10.0,
            stock: 100,
          );

          final model = ProductModel.fromEntity(product);

          expect(model.id, product.id);
          expect(model.name, product.name);
          expect(model.barcode, product.barcode);
          expect(model.price, product.price);
          expect(model.stock, product.stock);
        });

        test('toEntity should return Product instance', () {
          final entity = testProductModel.toEntity();

          expect(entity, isA<Product>());
          expect(entity.id, testProductModel.id);
          expect(entity.name, testProductModel.name);
        });
      });
    }
    ```

  <Tab title="Repository Tests">
    ```dart test/features/product/data/repositories/product_repository_impl_test.dart
    import 'package:flutter_test/flutter_test.dart';
    import 'package:fpdart/fpdart.dart';
    import 'package:hive/hive.dart';
    import 'package:mocktail/mocktail.dart';
    import 'package:billing_app/core/error/failure.dart';
    import 'package:billing_app/features/product/data/models/product_model.dart';
    import 'package:billing_app/features/product/data/repositories/product_repository_impl.dart';

    class MockHiveBox extends Mock implements Box<ProductModel> {}

    void main() {
      late ProductRepositoryImpl repository;
      late MockHiveBox mockBox;

      setUp(() {
        mockBox = MockHiveBox();
        // Inject mocked box via dependency injection
        repository = ProductRepositoryImpl();
      });

      group('getProductByBarcode', () {
        const testBarcode = '12345';
        const testProduct = ProductModel(
          id: '1',
          name: 'Test Product',
          barcode: testBarcode,
          price: 10.0,
          stock: 100,
        );

        test('should return Right(Product) when product exists', () async {
          // Arrange
          when(() => mockBox.values).thenReturn([testProduct]);

          // Act
          final result = await repository.getProductByBarcode(testBarcode);

          // Assert
          expect(result.isRight(), true);
          result.fold(
            (failure) => fail('Should not return failure'),
            (product) => expect(product.barcode, testBarcode),
          );
        });

        test('should return Left(CacheFailure) when product not found', () async {
          // Arrange
          when(() => mockBox.values).thenReturn([]);

          // Act
          final result = await repository.getProductByBarcode(testBarcode);

          // Assert
          expect(result.isLeft(), true);
          result.fold(
            (failure) => expect(failure, isA<CacheFailure>()),
            (_) => fail('Should not return product'),
          );
        });
      });
    }
    ```

### 3. Presentation Layer Testing

<Tabs>
  <Tab title="BLoC Tests">
    ```dart test/features/billing/presentation/bloc/billing_bloc_test.dart
    import 'package:flutter_test/flutter_test.dart';
    import 'package:bloc_test/bloc_test.dart';
    import 'package:fpdart/fpdart.dart';
    import 'package:mocktail/mocktail.dart';
    import 'package:billing_app/features/billing/presentation/bloc/billing_bloc.dart';
    import 'package:billing_app/features/product/domain/entities/product.dart';
    import 'package:billing_app/features/product/domain/usecases/product_usecases.dart';
    import 'package:billing_app/core/error/failure.dart';

    class MockGetProductByBarcodeUseCase extends Mock 
        implements GetProductByBarcodeUseCase {}

    void main() {
      late BillingBloc bloc;
      late MockGetProductByBarcodeUseCase mockUseCase;

      setUp(() {
        mockUseCase = MockGetProductByBarcodeUseCase();
        bloc = BillingBloc(getProductByBarcodeUseCase: mockUseCase);
      });

      tearDown(() {
        bloc.close();
      });

      const testProduct = Product(
        id: '1',
        name: 'Test Product',
        barcode: '12345',
        price: 10.0,
        stock: 100,
      );

      group('AddProductToCartEvent', () {
        blocTest<BillingBloc, BillingState>(
          'should add product to empty cart',
          build: () => bloc,
          act: (bloc) => bloc.add(const AddProductToCartEvent(testProduct)),
          expect: () => [
            predicate<BillingState>((state) {
              return state.cartItems.length == 1 &&
                     state.cartItems.first.product == testProduct &&
                     state.cartItems.first.quantity == 1;
            }),
          ],
        );

        blocTest<BillingBloc, BillingState>(
          'should increment quantity when adding existing product',
          build: () => bloc,
          seed: () => const BillingState(cartItems: [
            CartItem(product: testProduct, quantity: 1),
          ]),
          act: (bloc) => bloc.add(const AddProductToCartEvent(testProduct)),
          expect: () => [
            predicate<BillingState>((state) {
              return state.cartItems.length == 1 &&
                     state.cartItems.first.quantity == 2;
            }),
          ],
        );
      });

      group('ScanBarcodeEvent', () {
        blocTest<BillingBloc, BillingState>(
          'should add product when scan succeeds',
          build: () {
            when(() => mockUseCase('12345'))
                .thenAnswer((_) async => const Right(testProduct));
            return bloc;
          },
          act: (bloc) => bloc.add(const ScanBarcodeEvent('12345')),
          expect: () => [
            predicate<BillingState>((state) => state.cartItems.length == 1),
          ],
          verify: (_) {
            verify(() => mockUseCase('12345')).called(1);
          },
        );

        blocTest<BillingBloc, BillingState>(
          'should emit error when product not found',
          build: () {
            when(() => mockUseCase('99999'))
                .thenAnswer((_) async => const Left(CacheFailure('Not found')));
            return bloc;
          },
          act: (bloc) => bloc.add(const ScanBarcodeEvent('99999')),
          expect: () => [
            predicate<BillingState>((state) => 
              state.error != null && state.error!.contains('Product not found')),
          ],
        );
      });

      group('ClearCartEvent', () {
        blocTest<BillingBloc, BillingState>(
          'should reset to initial state',
          build: () => bloc,
          seed: () => const BillingState(cartItems: [
            CartItem(product: testProduct, quantity: 2),
          ]),
          act: (bloc) => bloc.add(const ClearCartEvent()),
          expect: () => [const BillingState()],
        );
      });
    }
    ```

## Testing Best Practices

<CardGroup cols={2}>
  <Card title="Arrange-Act-Assert" icon="1">
    Structure tests clearly:
    ```dart
    test('description', () {
      // Arrange: Set up test data
      const input = 'test';
      
      // Act: Execute the code
      final result = function(input);
      
      // Assert: Verify results
      expect(result, expected);
    });
    ```
  </Card>
  
  <Card title="Mock External Dependencies" icon="2">
    Use `mocktail` for mocking:
    ```dart
    class MockRepository extends Mock 
        implements ProductRepository {}
    
    when(() => mock.method())
        .thenAnswer((_) async => result);
    ```
  </Card>
  
  <Card title="Test Edge Cases" icon="3">
    Cover boundary conditions:
    - Empty lists
    - Null values
    - Error scenarios
    - Invalid input
  </Card>
  
  <Card title="Descriptive Test Names" icon="4">
    Use clear naming:
    ```dart
    test('should return Right(Product) when product exists')
    test('should emit error when barcode invalid')
    test('copyWith should update only specified fields')
    ```
  </Card>
</CardGroup>

### Common Mocking Patterns

<Tabs>
  <Tab title="Basic Mocking">
    ```dart Creating Mocks
    import 'package:mocktail/mocktail.dart';

    // Create mock classes
    class MockProductRepository extends Mock implements ProductRepository {}
    class MockHiveBox extends Mock implements Box<ProductModel> {}
    
    // In setUp
    late MockProductRepository mockRepo;
    
    setUp(() {
      mockRepo = MockProductRepository();
    });
    ```
  </Tab>
  
  <Tab title="Stubbing Methods">
    ```dart When/ThenAnswer
    // Stub method to return value
    when(() => mockRepo.getProduct('1'))
        .thenAnswer((_) async => const Right(product));
    
    // Stub to throw exception
    when(() => mockRepo.getProduct('invalid'))
        .thenThrow(Exception('Not found'));
    
    // Stub with dynamic return based on input
    when(() => mockRepo.getProduct(any()))
        .thenAnswer((invocation) async {
          final id = invocation.positionalArguments[0] as String;
          return id == '1' ? Right(product) : Left(failure);
        });
    ```
  </Tab>
  
  <Tab title="Verification">
    ```dart Verify Calls
    // Verify method was called
    verify(() => mockRepo.getProduct('1')).called(1);
    
    // Verify never called
    verifyNever(() => mockRepo.deleteProduct(any()));
    
    // Verify call order
    verifyInOrder([
      () => mockRepo.getProduct('1'),
      () => mockRepo.updateProduct(product),
    ]);
    ```
  </Tab>
</Tabs>

## Common Testing Patterns

### Testing Either Results

```dart Either Testing Helpers
// Helper to extract Right value
T getRightValue<L, T>(Either<L, T> either) {
  return either.fold(
    (l) => throw Exception('Expected Right but got Left: $l'),
    (r) => r,
  );
}

// Helper to extract Left value
L getLeftValue<L, T>(Either<L, T> either) {
  return either.fold(
    (l) => l,
    (r) => throw Exception('Expected Left but got Right: $r'),
  );
}

// Usage in tests
test('should return product', () async {
  final result = await useCase('123');
  final product = getRightValue(result);
  expect(product.id, '123');
});

test('should return failure', () async {
  final result = await useCase('invalid');
  final failure = getLeftValue(result);
  expect(failure, isA<CacheFailure>());
});
```

### Testing Equatable

```dart Equatable Tests
test('should support value equality', () {
  const item1 = CartItem(product: product1, quantity: 2);
  const item2 = CartItem(product: product1, quantity: 2);
  const item3 = CartItem(product: product1, quantity: 3);

  // Same values = equal
  expect(item1, equals(item2));
  expect(item1.hashCode, equals(item2.hashCode));
  
  // Different values = not equal
  expect(item1, isNot(equals(item3)));
});
```
